# Palindrome

## Defintion

A **palindrome** is a special type of word that is spelled the same backward or forward.
Palindromes can be words, sentences, groups of numbers, or even equations.

## Examples
- Eye
- Ewe
- Mom
- Dad
- Racecar
- Kayak
- Madman
- Redder
- Madam, I'm Adam.
- I did, did I?
- Nurses run!
- Step on no pets.
- 2002
- 1991
- 1771
- 100,001
